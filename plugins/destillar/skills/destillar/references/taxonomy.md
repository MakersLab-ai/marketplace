# IdeaCard-Enums

Referenz für die Felder einer IdeaCard (siehe Content-Ideas-Workflow in SKILL.md).

## Formate (`format`)

| Wert | Zielumfang |
|---|---|
| `article` | Langform, ca. 800–1500 Wörter — Blog, Deep-Dive |
| `post` | Kurzform, ca. 200–400 Wörter — Social, Newsletter-Block |
| `carousel` | 6–10 Slides mit je 1 Kernaussage — LinkedIn / Instagram |

## Storylines (`storyline`)

| Wert | Struktur |
|---|---|
| `how_to` | Problem → Schritte → Ergebnis |
| `contrarian` | Weit verbreitete Annahme → Gegenposition → Begründung |
| `case_study` | Situation → Intervention → Outcome |
| `framework_explainer` | Framework-Name → Komponenten → Anwendung |
| `story_hook` | Anekdote → Verallgemeinerung → Takeaway |
| `listicle` | Nummerierte Liste mit je kurzer Erklärung |

## Seed Reasons (`seed_reason`)

| Wert | Bedeutung |
|---|---|
| `framework_chain` | Methoden verknüpft zu einem Gesamtprozess |
| `cross_source_insight` | Konvergenz aus mehreren Quellen — hohe Konfidenz |
| `story_dense` | Viele Illustrationen → gut erzählbar |
| `unused_core` | Zentrale Einsicht ohne bisherige Verwendung |
| `topic_targeted` | Gezielt auf ein `topic` fokussiert |
